# 🔴 Live Streaming Simulation

## 📌 Project Overview
This project simulates a live streaming platform like YouTube Live using HTML, CSS, and JavaScript.

## 🚀 Features
- 🔴 Live streaming UI
- 👁 Dynamic viewer count
- 💬 Live chat simulation
- ⏱ Streaming timer
- 🌙 Dark theme UI
- 📱 Responsive design

## ▶️ How to Run
1. Download the project files
2. Add a video file named "video.mp4" in the same folder
3. Open index.html in your browser
